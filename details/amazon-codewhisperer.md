# Amazon CodeWhisperer

**An AI-powered code generation service by AWS that provides real-time code suggestions and security scanning directly in your IDE.**

## Features
- **Real-time code suggestions**: Generates whole lines and full functions based on comments and existing code.
- **Security scanning**: Built-in security vulnerability detection that scans generated code for common issues.
- **AWS integration**: Deep integration with AWS services, making it ideal for cloud-native development.

## Pricing
Amazon CodeWhisperer offers a free Individual tier with unlimited code suggestions. The Professional tier is $19/user/month with additional security and admin features. For more details, visit [aws.amazon.com/codewhisperer](https://aws.amazon.com/codewhisperer).
