# autocomplete.sh

Autocomplete.sh uses GPT models to suggest bash commands in real-time, parsing history and context for precise terminal completions. It's lightweight, privacy-first, and extensible via custom prompts. Ideal for 2026 shell warriors, it edges out Fig.io by running fully offline with local LLMs.
