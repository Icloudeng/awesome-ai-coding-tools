# Autodoc

Autodoc leverages LLMs to generate comprehensive docs, diagrams, and changelogs from any codebase, supporting Markdown, PlantUML, and API specs. It scans repos holistically for architecture overviews and function docs. In 2026, it's the go-to open-source alternative to Swimm for teams ditching manual doc maintenance.
