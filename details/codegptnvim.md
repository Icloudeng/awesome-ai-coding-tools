# CodeGPT.nvim

Neovim plugin for ChatGPT integration, providing code generation, refactoring, and explanations directly in the editor with custom prompts. Supports multiple models and session persistence. Vim enthusiasts in 2026 pair it with LazyVim configs over native Copilot for lightweight, open extensibility.
