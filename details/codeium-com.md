## Features

- Supports over 70 programming languages out of the box.
- Integrates with VS Code, JetBrains, Neovim, Emacs, and more.
- Fast autocomplete with context-aware suggestions trained on permissively licensed code.
- AI-powered code search across your repository.
- Free for individual developers with no usage limits.

## Pricing

Codeium offers free and paid tiers:

- **Free Plan**: Unlimited autocomplete for individual developers.
- **Teams Plan**: Advanced features including team-wide context awareness and admin controls.
- **Enterprise Plan**: Self-hosted deployment, SSO, and custom model training.

For more details, visit the [Codeium Pricing](https://codeium.com/pricing) page.
