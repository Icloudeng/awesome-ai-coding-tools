# Cursor

**An AI-first code editor built on VS Code, featuring intelligent code generation, multi-file editing, and natural language commands for coding tasks.**

## Features
- **AI-powered code generation**: Generate code from natural language prompts directly in the editor.
- **Multi-file editing**: Make changes across multiple files simultaneously with AI assistance.
- **Codebase-aware context**: Understands your entire project for more relevant suggestions.

## Pricing
Cursor offers a free tier with limited AI usage. Pro plans start at $20/month. For more details, visit [cursor.com/pricing](https://cursor.com/pricing).
