## Overview

GitHub Codespaces offers fully cloud-hosted development environments that spin up in seconds, pre-configured for AI tools and custom dependencies.

## Features

- Instant provisioning of secure, isolated VS Code instances in browser or desktop.
- Devcontainer support for reproducible environments with languages, tools, and extensions.
- Real-time collaboration, live share, and pair programming capabilities.
- Integration with GitHub Actions for CI/CD, pull requests, and workflows.
- Scalable compute including GPU options for ML workloads.
- Persistent storage and forwarding ports for local testing.

## Use Cases

- Remote team development without local setup hassles.
- Onboarding new developers with pre-configured repos.
- Prototyping and testing across environments.

## Pricing

Free: 60 core-hours/month (2-core machine). Paid: $0.18/core-hour beyond free tier. Storage $0.07/GB-month. Enterprise plans available.
