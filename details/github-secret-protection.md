## Overview

Prevents secret leaks with AI-driven detection during development workflows.

## Features

- Semantic analysis for tokens, keys, and certs.
- IDE extensions for instant feedback (VS Code, JetBrains).
- Automatic blocking of pushes with secrets.
- Custom detector training.
- Audit logs and remediation guidance.

## Use Cases

- Enterprise compliance in CI/CD pipelines.
- Solo devs securing side projects.
- Migrating from legacy scanning tools.

## Pricing

Available in GitHub Advanced Security: Free for public repos; $49/user/month for private (Team); Enterprise custom.
