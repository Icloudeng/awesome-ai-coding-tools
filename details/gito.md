## Overview

Flexible AI-powered PR reviewer for Git workflows.

## Features

- LLM-agnostic: Use OpenAI, Anthropic, or local models.
- GitHub Actions integration for CI reviews.
- CLI for local testing and batch analysis.
- Jira/Linear ticket linking and summaries.
- Configurable review depth and style.

## Use Cases

- Automating open-source project reviews.
- Custom enterprise review pipelines.
- Offline debugging of code changes.
