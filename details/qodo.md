## Overview

Qodo is a comprehensive AI code quality platform originally known as CodiumAI, focused on code review and validation throughout the development lifecycle.

## Features

- AI Code Review Platform: Enterprise-grade code review across the SDLC
- Pull request review: Git integration ([Qodo Git](https://www.qodo.ai/features/qodo-git/))
- Local code review: IDE plugin ([Qodo IDE](https://www.qodo.ai/features/qodo-ide/))
- Agentic quality workflows: CLI Tool ([Qodo CLI](https://www.qodo.ai/features/qodo-cli/))
- Full codebase intelligence: Context Engine ([Context Engine](https://www.qodo.ai/features/qodo-context-engine/))
- Rules System: Discover, enforce, and maintain rules ([Qodo Rules](https://www.qodo.ai/qodo-rules/))

## Pricing

Free for open source projects ([Open Source](https://www.qodo.ai/solutions/open-source/)). Paid plans available; details at [pricing](https://www.qodo.ai/pricing/).

## Use Cases

- Context-aware PR validation
- Enforcing coding standards and compliance
- Scaling code review for large, multi-repo teams
