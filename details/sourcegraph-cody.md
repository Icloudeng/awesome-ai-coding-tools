# Sourcegraph Cody

**An AI coding assistant by Sourcegraph that leverages entire codebases for context-aware code generation, explanations, and refactoring.**

## Features
- **Codebase-wide context**: Uses Sourcegraph's code graph to understand your entire codebase, not just the open file.
- **Multi-LLM support**: Choose between Claude, GPT, Gemini, and other models depending on your needs.
- **IDE integration**: Available as extensions for VS Code, JetBrains IDEs, and Neovim.

## Pricing
Cody offers a free tier for individual developers. Pro plans start at $9/month with unlimited AI completions. For more details, visit [sourcegraph.com/pricing](https://sourcegraph.com/pricing).
