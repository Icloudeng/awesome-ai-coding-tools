# Windsurf

**An agentic AI-powered IDE by Codeium, designed for deep codebase understanding and collaborative coding with AI flows.**

## Features
- **Cascade AI flows**: Multi-step agentic AI that can reason, plan, and execute complex coding tasks across your codebase.
- **Deep codebase awareness**: Indexes your entire project for contextually accurate suggestions and edits.
- **Inline code generation**: Generate, refactor, and debug code with natural language prompts directly in the editor.

## Pricing
Windsurf offers a free tier with basic AI features. Pro plans start at $15/month with unlimited AI usage. For more details, visit [windsurf.com/pricing](https://windsurf.com/pricing).
